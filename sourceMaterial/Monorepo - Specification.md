# Monorepo Specification

**Objective**:  
Set up a **monorepo** architecture containing two applications:

- A **React Native** mobile app

- A **ReactJS** web app

# **Requirements**:

- Implement a **shared module** to fetch user from a remote server.

- Fetch user data from [Dummy Users](https://dummyjson.com/docs/users).

- Address **CORS issues** appropriately.

# **Expectations:**

- Use appropriate tooling to manage the monorepo.

- Design the shared code for reusability, maintainability, and testability.

- Employ sound architectural decisions and adherence to best practices.
